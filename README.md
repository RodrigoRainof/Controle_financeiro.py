# Sistema de Controle Financeiro

Projeto desenvolvido em Python com SQLite para gerenciamento de receitas e despesas.

##  Funcionalidades
- Cadastro de receitas e despesas
- Listagem de transações
- Cálculo de saldo total
- Filtro por mês

##  Tecnologias
- Python
- SQLite
- VS Code

##  Como executar

```bash
python main.py
